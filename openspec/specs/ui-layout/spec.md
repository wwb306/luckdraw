## Requirements
### Requirement: Layout Responsiveness
The system SHALL adjust the layout of the lucky draw workspace to ensure optimal visibility of the primary action buttons.

#### Scenario: Center Start Button
- Given a prize is selected in the lucky draw workspace
- And no draw is currently in progress
- Then the "Start Draw" button MUST be positioned in the vertical center of the content area
- And the decorative elements (icons, secondary text) MUST adjust their height to accommodate this centering
